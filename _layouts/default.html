1. 창이름 변경하기

녹스앱플레이어의 멀티플레이어를 실행한 뒤 플레이어 창 이름을 지정합니다. 한글,영문,숫자,공백,특수문자 등을 사용해 만드셔도 됩니다.

* 이 때 지정된 창이름을 매크로 실행화면에 입력해주세요.

2. 창 해상도 변경하기

사용자지정 선택 후 가로 800, 세로 450, DPI 160으로 설정값을 수정해주세요.

3. 창 크기 고정하기

앱플레이어를 시작합니다. 그 후 "고정 된 윈도우 사이즈", "기존 위치 및 사이즈 유지"를 설정 해주셔야 정상적으로 동작합니다

* 창크기 인식 오류 시 "고정 된 윈도우 사이즈", "기존 위치 및 사이즈 유지"를 해제한뒤 앱플레이어를 재실행 해주시기 바랍니다.

4. 가상키 투명도 조절

가상키를 지정하여 사용하는 경우, 투명도를 반드시 0%로 해주셔야 합니다.

1. 창이름 변경하기

모모앱플레이어의 멀티플레이어를 실행한 뒤 플레이어 창 이름을 지정합니다. 한글,영문,숫자,공백,특수문자 등을 사용해 만드셔도 됩니다.

* 이 때 지정된 창이름을 매크로 실행화면에 입력해주세요.

2. 창 해상도 변경하기

사용자 입력 체크 후 가로 800, 세로 450, DPI 160으로 설정값을 수정해주세요.

3. 창 크기 고정하기

창크기 고정을 해제한 뒤 앱플레이어를 시작합니다. 그 후 창크기 고정을 해주셔야 정상적으로 동작합니다

* 현재 모모 1.1.4 버전에서는 창크기 고정을 해제해야합니다.

* 창크기 인식 오류 시 고정을 해제한뒤 앱플레이어를 재실행 해주시기 바랍니다.

1. 매크로 로그인

ID : test, PW : test 로 로그인 해주세요. (Save 체크)

* 접속자 초과가 뜨는 경우 잠시 후 재시도 하시면됩니다.

* 알약, 아바스트 등 일부 백신의 오진으로 인한 문제로, 제외 항목에 폴더를 추가해주세요.

2. 매크로 설정

창이름에 멀티플레이어에서 넣은 이름을 입력해주세요.

원하는 모드를 선택한 뒤 세부적인 설정을 한 뒤, START 버튼을 눌러 매크로를 시작합니다.

* 각 메뉴 우측의 ? 버튼을 눌러 설명을 반드시 확인하세요.

3. 스케줄러 방식 설정

이동할 위치와 세부 옵션을 입력 한 뒤 추가 버튼을 눌러 스케줄 목록에 추가해주세요.

* 시간 설정 방식에서 시간이 겹치는 경우 오동작의 소지가 있습니다. 주의하세요!

1. 압축풀기

원하는 경로에 압축을 풀어주세요. Main 폴더는 절대 건드리지 마세요.

* 알약, 아바스트 등 일부 백신의 오진으로 인한 문제로, 제외 항목에 폴더를 추가해주세요.

2. 부계정 폴더 생성

Subs 폴더 아래에 운용할 만큼의 폴더를 만들어주세요.

* 매크로 설정값은 각 폴더마다 개별적으로 적용됩니다.

* 필요에 따라 Subs 안의 폴더를 옮겨 백업/복원이 가능합니다.

3. 업데이트 및 매크로 일괄 실행

Update.bat 파일을 오른쪽 클릭 한 뒤 "관리자 권한으로 실행"을 눌러 실행합니다. 이 과정에서 뜨는 검은 화면은 가만히 두시면 됩니다.

* 원활한 매크로 실행을 위해 반드시 로그인 화면의 Save를 체크해야합니다. Save 체크는 다음 실행부터 적용됩니다.
